Lab 4 - Angular
================

This folder contains your lab 4 work:

- `task1/` – place your completed official Angular "Learn Angular" tutorial project here.
- `task2/online-store/` – standalone Angular project for the "Online Store" assignment.

Task 2 (online-store) – how to run
----------------------------------

1. Open a terminal in this folder:

   - `cd lab4/task2/online-store`

2. Install dependencies:

   - `npm install`

3. Start the dev server:

   - `ng serve`

4. Open the app in your browser at:

   - `http://localhost:4200`

Make sure you have Node.js, npm, and the Angular CLI installed globally before running these commands.
